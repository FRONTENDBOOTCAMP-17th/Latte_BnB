## 3-1. 인증 ��크 코드가 4곳에서 거의 동일하게 복사되어 있어요 (중요도: 높음)

### 문제

`wishlist/index.js`(7~29줄), `admin/index.js`(19~51줄), `admin/add/index.js`(20~40줄)에서 거의 동일한 인증 체크 코드가 복사-붙여넣기 되어 있어요. `src/main.js`(42~72줄)에도 유사한 `checkAuth` 함수가 있어요.

### 해결 방법

**Step 1.** `src/auth.js` 파일을 새로 만드세요.

이 파일은 인증 관련 로직을 **한 곳에 모아두는 역할**이에요. 이걸 "관심사 분리"라고 해요 — 인증은 인증 모듈이, UI는 UI 모듈이 담당하는 거예요.

```js
// src/auth.js (신규 파일)
import constants from './constants.js';

/**
 * 일반 사용자 인증 체크
 * 토큰이 없거나 유효하지 않으면 로그인 페이지로 이동하고 null 반환
 * 유효하면 사용자 프로필 데이터 반환
 */
export async function checkUserAuth({ redirectTo = '/login/' } = {}) {
  const token = localStorage.getItem('accessToken');

  if (!token) {
    location.replace(redirectTo);
    return null;
  }

  try {
    const res = await fetch(`${constants.API_BASE_URL}/me/profile`, {
      method: 'GET',
      headers: {
        Authorization: `Bearer ${token}`,
      },
    });

    if (!res.ok) {
      localStorage.removeItem('accessToken');
      location.replace(redirectTo);
      return null;
    }

    const { data } = await res.json();
    return data;
  } catch (error) {
    localStorage.removeItem('accessToken');
    location.replace(redirectTo);
    return null;
  }
}

/**
 * 관리자 인증 체크
 * admin_token이 없거나 유효하지 않으면 관리자 로그인 페이지로 이동
 */
export async function checkAdminAuth() {
  const token = localStorage.getItem('admin_token');

  if (!token) {
    cleanupAdminSession();
    location.replace('/admin/login/');
    return null;
  }

  try {
    const res = await fetch(`${constants.API_BASE_URL}/me/profile`, {
      method: 'GET',
      headers: {
        Authorization: `Bearer ${token}`,
      },
    });

    if (!res.ok) {
      cleanupAdminSession();
      location.replace('/admin/login/');
      return null;
    }

    const { data } = await res.json();
    return data;
  } catch (error) {
    cleanupAdminSession();
    location.replace('/admin/login/');
    return null;
  }
}

function cleanupAdminSession() {
  localStorage.removeItem('admin_token');
  localStorage.removeItem('admin_info');
}
```

## 3-8. RoomCard.js에서 가격에 천 단위 콤마가 없어요 (중요도: 낮음)

### 문제

`src/RoomCard.js` 108줄에서 가격을 표시할 때 `toLocaleString()`을 사용하지 않아서 "₩ 80000 / 1박"처럼 표시돼요.

### 해결 방법
